# Yemen Health Cluster Website - Vercel Ready

This package is ready to publish for free on Vercel.

## Files
- `index.html` - main website
- `vercel.json` - simple static routing config
- `logo-placeholder.svg` - replace with the approved official logo
- `team-data.js` - optional helper notes for team updates

## Publish on Vercel
1. Go to Vercel and sign in.
2. Click **Add New** -> **Project**.
3. Choose **Browse** or drag this whole folder.
4. Deploy.
5. Vercel will publish it automatically.
